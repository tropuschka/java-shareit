# java-shareit
Веб-сервис для аренды предметов. Позволяет создавать карточки предметов для сдачи в аренду, искать и арендовать предметы на определенны срок и оставлять комментарии.

## Задачи
Реализация бизнес-логики, настройка валидации данных, настройка и подключение базы данных, написание юнит-тестов.

## Технологии
Spring Boot, REST, Maven, Lombok, PostgreSQL, JUnit.
